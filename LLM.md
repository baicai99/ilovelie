# 我爱撒谎
## 描述
为了应对能写好代码被裁员的问题，我打算开发一个对注释撒谎的插件，使得代码只有“我”可以维护，应对被开除的风险，目前只针对vscode的插件。

## 功能
- 手动替换：手动输入想要撒谎的注释
- 字典替换：从注释中遍历字典关键字，然后替换掉注释
- 隐藏：隐藏注释
